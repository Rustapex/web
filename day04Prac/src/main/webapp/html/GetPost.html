GET / POST

GET 요청 A 태그 , URL 주소 줄에 직접 입력, form 태그 method="get" (default : get)
서버에 간단한 정보의 데이터를 보낼 때에 사용함 (주소줄에 노출됨) , 데이터 형식, 데이터 길이 제한되어있음
단순 조회 기능
캐시 지원

get 요청

post 요청 : 서버의 상태가 바뀌는 작업(등록, 변경, 삭제)
form 태그에서 method="post" 요청해야함
사용자의 다양한 데이터를 서버로 보낼 수 있다.
사용자의 데이터가 노출 X 요청 body에 담겨서 보내짐. text img 영상 보낼 수 있다.
