	요청			응답
	- get		-get (서버가 가지고 있는 정보를 단지 제공해 주는 서비스)
	- post		-post (
	
요청 구분 예시
get 	: a태그, url 요청, form tag( method = "get", method 미지정시 default) => server get 요청을 인식 
post 	: form tag(method = "post")										=> server post 요청을 인식

get 요청 : 사용자가 보내는 데이터를 header에 담는다. url노출, data type과 크기가 정해져 있음
post 요청 : 사용자가 보내는 데이터를 요청 body에 담는다. 노출 x, data 형식, 크기 제한 x

요청 구분 이유 
효율적인 서버의 응대
get : 가벼운 용청

service -> request 에 요청 정보 확인 ->
	get -> doGet, post doPost
	
요청을 구분하는 것이 일반적임.
post : 무거운 요청
